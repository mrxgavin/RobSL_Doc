Introduction

Welcome to RobSL, your comprehensive tool for exploring synthetic lethality in cancer research. RobSL integrates large-scale functional loss data from cancer cell lines with multi-omics data, enabling the identification of robust synthetic lethal gene pairs. This guide will help you navigate and utilize RobSL effectively.

Home Page

The Home page provides an overview of RobSL and its capabilities in cancer research. It also includes a step-by-step usage guide and displays dataset metrics.

RobSL Page

On the RobSL page, you can select the cancer type and driver gene for analysis. You can input candidate genes manually, upload a file, select from KEGG pathways, or choose all common genes in the dataset. Additionally, you can apply filters based on gene expression, copy number variation, and mutations.

Annotation Page

The Annotation page allows you to submit the final result file generated from the RobSL page for annotation. The annotated file will include data from known databases, protein interaction networks, and drug response data.

Visualization Page

The Visualization page enables you to upload the annotated result file and explore the results using various visualizations such as scatter plots and PPI networks.

Synthetic Lethality - Drug Network Page

On this page, you can upload the annotated result file with drug data and construct a synthetic lethality-drug network. This helps in discovering potential therapeutic strategies.

Getting Started

Install Dependencies: Ensure you have the required Python packages installed. Use the following command: 

pip install streamlit pandas numpy matplotlib seaborn networkx scipy base64 re tqdm

Run the Application: Navigate to the directory containing the RobSL scripts and run the application using: 

streamlit run Home.py
